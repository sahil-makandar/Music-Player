## Getting Started

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn

### Installation
1. Clone the repository
```bash
git clone https://github.com/sahil-makandar/Music-Player.git
```

2. Install dependencies
```bash
npm install
```

3. Start the development server
```bash
npm start
```
